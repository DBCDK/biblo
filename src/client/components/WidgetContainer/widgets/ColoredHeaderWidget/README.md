# ColoredHeaderWidget

Renders a header with a background color and some text.

Usage: 
```javascript
{
  widgetName: 'ColoredHeaderWidget',
  widgetConfig: {
    color: '#f3f4f5', // Optional
    text: 'biblo.dk er mega sejt!', 
    title: 'Velkommen!'
  }
}
```

## NB.
This widget was made for the front page and may not work elsewhere.
