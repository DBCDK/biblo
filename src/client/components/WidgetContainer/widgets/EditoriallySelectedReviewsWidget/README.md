# EditoriallySelectedReviewsWidget

This widget renders editorially selected reviews

Usage: 
```javascript
{
  widgetName: 'EditoriallySelectedReviewsWidget',
  widgetConfig: {
    title: 'Super titel!',
    showTitle: true,
    backgroundColor: '#FF00FF',
    backgroundImageUrl: 'http://i.imgur.com/ls2fKsP.jpg',
    reviewIds: [
      1,
      2,
      3,
      4,
      5,
      6
    ]
  }
}
```
