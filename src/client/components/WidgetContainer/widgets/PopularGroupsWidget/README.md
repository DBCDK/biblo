# PopularGroupsWidget

This widget renders a grid of popular groups.

Usage: 
```javascript
{
  widgetName: 'PopularGroupsWidget',
  widgetConfig: {
    title: 'Super titel!',
    showTitle: true,
    backgroundColor: '#FF00FF',
    backgroundImageUrl: 'http://i.imgur.com/ls2fKsP.jpg',
    groupsToLoad: 20
  }
}
```
